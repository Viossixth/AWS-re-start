**Accessing the terminal session**

To access a new terminal, i chose the + icon and selected New Terminal. A new tab opens with the terminal connected. To confirm the current working directory, I used te "pwd" command. 
To confirm the file I had created I used the "ls" command.

I then confirmed the version of python installed. After confirmation, I returned back to my created Python file. I entered the code " print("Hello,World") " and saved the file. 
This code when ran will output Hello,World, and to confirm this I pressed the Run button and it worked perfectly.
