## Working with Amazon S3 
The main aim of this lab is to configure and Amazon S3 bucket with the aim of sharing images captured by a photographer. The images will be sent to a media company. The bucket should also send an email should any changes be made to the content os the bucket.
<img width="534" height="362" alt="Architecture" src="https://github.com/user-attachments/assets/99e2dc10-abbc-4b94-8f12-d0836b39eff0" />

To being, log into the AWS Management console, and go to EC2. Look at the instances, select CLI Host and connect to it via EC2 Instance Connect. Configure AWS on the new connection using the command 'aws configure'.
