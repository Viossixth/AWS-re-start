## Lambda

Lambda is a serverless compute service offered by AWS. It allows developers to run code without maintaining the underlying infrastructure. In this case serverless means without setup and maintenance of serverless of EC2 instances.
Lambda is a plug and play service, allowing rapid deployment of code. Rapid scaling is the main use case of Lambda. It supports Noed.js,Java,C#, Ruby and Python among others as the run-time languages.

**Use Cases**

* Interactive web and mobile backends
* Batch data processing
* Real-time data processing
* Generative AI

**Benefits of AWS Lambda** 

* Increase developer agiglity
* Boost application Performance
* Help Strengthen overall security posture
* Drive cost efficiency
