##Databases

Databases are  places to hold, sort and keep informtation on a computer. A database is a large sorting pile for information. They can contain thousands of files, and can aid in sorting, filtering and maintaining file. They can contains, logs,documents, videos or pictures. There are two main database types. Relational and Non-relational known as No-SQL database. SQL is Structured Query Language used to interact with Relational Databases such as MySQL and Amazon Aurora.

The relational databases offered by AWS are Amazon RDS, Amazon Aurora, MySQL and PostgreeSQL. Relational databases have tables, rows and columns and information is stored in a row in one event. A database may have multiple rows and columns within a table and many more tables within a Database. The information within the columns in one table may also be available in another column in another table. This forms the relational part of the database. Primary and Foreign Keys are used to accomplish this. The information stored in relational databases might be customer information, postal addresses and customer data from the main platforms such as a Shopping website. 

Non-relational databases collects and stores unstructured and semi-structured data. This includes may include plain text or images,user information within a document and . Non-relational databses uses documents as structured data records in a non-tubular format in JSON to store the information. Non-relational databases have a flexible schema, where one can determine the relationship or elements. Amazon offers Amazon DynamoDB and Amazon DocumentDB as a non-relational databases.

Other database types are Graph databases and In-Memory Database. Graph databases uses nodes and relationships to show indicate the relationships between data.Amazon offers Amazon Neptune as a graphical database. In-memory databases store data in RAM for super fast access to data. Amazon offers Amazon Elasticache and Amazon MemoryDB as In-Memory Databases. Amazon Timestream is an example of a Time series Database offered by AWS. Amazon QLDB is a ledger database with an immutable and cryptographically verifiable record book. Used primarily for audit trails and financial records. Amazon Redshift is used for data warehousing and used for designed analytics and reporting. They're used for business intelligence.
