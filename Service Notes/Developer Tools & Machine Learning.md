CloudFormation is an Infrastructure as Code service that allows you to deploy a stack of services using a template. The template is wriiten in JSON or YAML and is used to create consistent and repeatable deployments.

* Scale your infrastructure

* Automate resource management

**Use cases**

* Manage infrastructure with DevOps
* Scale productions stacks
* Share best practices

## AWS CodeBuild

AWS CodeBuild is a  serverless Continious Integration service that compiles source code, run tests and produces software packages for deployment. It fully managed, enabling developers to build specs with AWS integration with a Pay as you go model.

**Benefits**

 * No setup or maintenance
 * Scale capacity
 * Pay by use
 * Prepackaged build environments

**Use cases**

* Automate CI/CD pipelines
* Remove the complexity of managing build servers
* Build Source code hosted on GitHub

## AWS CodeCommit

AWS CodeCommit is a source control service for Git-based repositories. It enables collaboration in a scalable and highly available environment , with security features built in hand and integrated with AWS developer tools.

**Benefits**

* Remove the need to manage source control servers
* Customize user-specific access
* Keep repositories highly available
* Maintain your repositories

**Use cases**

* Collaborate on code
* Use your existing tools
* receive notifications and custom scripts
* create up to 5 000 repositories by default

AWS CodeDeploy is a service to automate software deployments to other compute services such as Lambda and EC2. It reduces downtime and errors during application updates. CodeDeploy supports multiple compute platforms, has built in deployment strategies and has automatic tollbacks.

* Monitor fleet health
* Launch and track applications
* Integrate with your existing software

**Use cases**

* Automate deployments to remove manual operations
* Deploy to many hosts
* Use advanced deployments techniques
* Monitor health and roll back

## Amazon CDK

AWS Cloud Development Kit is a framework that allows you to define cloud infrastructure using TypeScript, Python, Java, C# or Go. It creates templates making it easier to write, share and manage infrastructure as code.
It is code-based infrastructure, has multi-language support and CI/CD friendly.

<img width="1788" height="462" alt="image" src="https://github.com/user-attachments/assets/b05e1665-b7dd-41c7-af45-b052d8f8c16c" />

**Benefits**

* Expressive power of familiar programming
* Simplify your AWS onboarding
* Design and share reusable components
* Build applications, write runtie code, and define resources.

**Use cases**

* Improve infrastructure and business logic
* Provision infrastructure patterns faster
* Automate AWS service provisioning with construct hub
* Write applications using tools built for the cloud.
