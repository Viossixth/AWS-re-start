## Security Services 

Amazon uses Identity Access Management to control access to resources. Users, Groups, Roles and Policies can be configured using IAM to meet business and organizational needs.With IAM credentials can be configured for identity verification. IAM enforces the principle of least privilledge to protect resources and security best principle  like MFA  should be configured to best protect resources and verify identities.

Amazon CloudWatch is another security service which montors a clients AWS environment and application in real time. It offers metrics, logs, Alarms, Dashboards and events and automations. Amazon CloudTrail is a logging and auditing service offered by AWS. This tracks each API call made within the clients AWS environment. CloudTrail helps with operational and risk auditing, governance and compliance.

AWS Shield is DDOS protection service for your applications. It protects them from cybersecurity attacks known as Distribution of Denail of Service known as DDOS attacks where an application is flooded with traffic to disrupt operations. Amazon GuardDuty is threat detection services by AWS. It continously analyses the AWS environment for any anomalies. Amazon Inspector is a vulnerability management service. It inspects your AWS Environment to see vulnerabilities and unexpected network exposures. Amazon TrustedAdvisor is a service used to optimise a clients AWS Environment.

AWS Network Firewall is a network level firewall to protect virtual private clouds. AWS Firewall Manager is used to centrally manage firewalls and security protocols among multiple AWS accounts and resources. Amazon Web Application Firewall allows control of http and https traffic and protects against SQL-Injection, cross-site-scripting and bots.
