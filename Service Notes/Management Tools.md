# AWS Management Tools

## CloudFront

CloudFronti is an AWS Management tools that allows deployment of stack resources as templates. It allows for control of various services from one vantage point, enabling developers and businesses to deploy many services at once. 
You can either use a template or define your own template for your stack resources. CloudFront allows for easy scalling, consistency with stack resources deployed due to a template being used and a fast deployment speed. CloudFront is an Infrastructure as Code service and works within an AWS. It can also repeat deployments across regions and accounts.
